# ✅ To Do App

## 📌 Overview

A simple and clean task management application built with Flutter.
It helps users organize daily tasks efficiently.

---

## 🚀 Features

* ➕ Add tasks
* ✔️ Mark tasks as completed
* ❌ Delete tasks
* 📋 View all tasks

---

## 🛠️ Technologies

* Flutter
* Dart

---

## ▶️ How to Run

1. Clone the repository
2. Run `flutter pub get`
3. Run the app
